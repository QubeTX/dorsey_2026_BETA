# dorsey_2026_BETA

This project is a manual rewrite or a re-engineering of the website LeonLeeDorsey.com. The original website was made with Squarespace website builder, and I'm looking to rebuild it entirely with code. This was originally for one of my clients, so I did build the original website, and now we're looking for more customization and flexibility.

I have in the OLD_DORSEY folder all of the Dorsey content, like the assets that were used on that Squarespace site, as well as the crawls or a basic website crawl of each of the pages and the text content. You should also have a visual description of the website, a visual overview of the website as well within that folder.

Our goal is to make this as modern as possible while still visually representing the jazz artist as he should be represented as a jazz musician. Should be designed with time-testsed Bauhaus principles / precision, and authentic New York soul.

AI Agents accessing this repo for development should have access to the FireCrawl MCP. That MCP server should allow you to crawl, scan, and search for more information or context on the live Squarespace site for context of reference while you're developing the new hardcoded site.
